# Ethical Use Statement

This repository documents an academic cybersecurity laboratory exercise focused on analysing system behaviour under high network load conditions.

All experiments described in this project were performed:

- In a controlled virtual lab environment
- On authorised systems
- For educational purposes only

The purpose of this work is to understand how network traffic patterns can affect system performance and service availability in order to improve defensive cybersecurity knowledge.

The techniques discussed should not be used to disrupt or interfere with systems without explicit permission.

The author does not endorse or support unauthorised use of these methods.
